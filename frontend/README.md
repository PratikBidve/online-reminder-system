# Frontend (React + Vite)

## Setup
1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the development server:
   ```sh
   npm run dev
   ```

## Features
- User authentication UI
- Reminder creation and listing
- Responsive design

## Folder Structure
- `/src` — Source code
- `/src/pages` — Main pages (Login, Signup, Reminders)
- `/src/components` — Reusable components
